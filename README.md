# RCB Automation LLC - Premium Vercel Version

## Open in VS Code
1. Unzip this folder
2. Open the folder in VS Code
3. Open a terminal
4. Run:
   npm install
   npm run dev

## Deploy to Vercel
- Push to GitHub and import into Vercel, or upload the folder contents
- Build command: npm run build
- Output directory: dist

## Main files
- src/App.jsx
- src/main.jsx
- src/index.css
